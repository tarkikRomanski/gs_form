== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/gs_form` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'GS Form' screen in WordPress
1. Use the Settings->GS Form screen to configure the plugin
1. Generate your own form or an existing
1. Just insert the right place shortkod
